<h1>CueLite</h1>

<p>Professional live show playback and cue control software for macOS.</p>

<h2>Features</h2>

<ul style="display:inline-block; text-align:left">
	<li>Cue list with transport controls</li>
	<li>Per-project settings (fade duration, audio import handling)</li>
</ul>

<h2>Usage</h2>

<ul style="display:inline-block; text-align:left">
	<li>Launch the app to see the Start Menu. Create or open a project to load the main CuePlayer window.</li>
	<li>Settings and About windows are available from the app menu.</li>
</ul>
